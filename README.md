# O'Foot
----

## App aiming at improving amateur soccer clubs' management
O'Foot provides a simple yet powerful interface to manage your soccer club with ease.  

-----

## Installation & Usage
- Install dependencies :
  - `composer install`
  - `yarn install`

- Run the Symfony & Webpack servers :
  - `symfony serve -d`
  - `yarn start`
